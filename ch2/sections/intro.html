Chapter 2

Elliptic curves as cryptographic groups

The purpose of this chapter is to introduce elliptic curves as they are used in cryptography. Put simply, an elliptic curve is an abstract type of group.

本章的目的是介绍椭圆曲线在密码学中的应用。简单来说，椭圆曲线是一种抽象的"群"。
什么是群？ 群就是一个集合，集合中的元素定义了一种二元运算（比如叫"加法" $+$）。这个运算必须满足几个条件：
① 封闭性：任意两个元素运算后还在集合里；
② 结合律：$(a + b) + c = a + (b + c)$；
③ 单位元：存在一个"零元" $0$，使得 $a + 0 = a$；
④ 逆元：每个元素 $a$ 都有一个逆元 $-a$，使得 $a + (-a) = 0$。
如果还满足 交换律 $a + b = b + a$，就叫交换群（或阿贝尔群）。
例子：整数集合 $\mathbb{Z}$ 配上普通加法就是一个交换群。

Perhaps a newcomer will find this abstractness apparent immediately when we insist that to understand elliptic curve groups in cryptography, the reader should be familiar with the basics of finite fields $\mathbb{F}_q$. This is because, more generally, elliptic curves are groups which are defined on top of (over) fields. Even though elliptic curve groups permit only one binary operation (the so called group law), the operation itself is computed within the underlying field, which by definition permits two operations (and their inverses). 新手可能马上就会觉得这很抽象——我们要求读者先熟悉有限域 $\mathbb{F}_q$ 的基础知识。这是因为椭圆曲线是定义在域上的群。虽然椭圆曲线群只允许一种二元运算（所谓的"群律"），但这个运算本身是在底层的域中计算的，而域按定义允许两种运算（及其逆运算）。对于一般的域 $K$，椭圆曲线 $E$ 的群元素是坐标 $(x, y)$ 来自 $\bar{K}$（$K$ 的代数闭包）的点，这些点满足 $E$ 的仿射曲线方程：
什么是域？ 域是一个集合，定义了加法和乘法两种运算，满足我们熟悉的算术规则（交换律、结合律、分配律），有加法单位元 $0$ 和乘法单位元 $1$，每个元素有加法逆元，每个非零元素有乘法逆元。常见的域：有理数 $\mathbb{Q}$、实数 $\mathbb{R}$、复数 $\mathbb{C}$。

什么是有限域 $\mathbb{F}_q$？ 元素个数有限的域。$q$ 必须是素数的幂，即 $q = p^n$。
• 当 $q = p$ 是素数时，$\mathbb{F}_p = \{0, 1, 2, \ldots, p-1\}$，就是模 $p$ 的整数，加法和乘法都取模 $p$。例如 $\mathbb{F}_{11}$ 中，$7 + 6 = 13 \mod 11 = 2$。
• 当 $q = p^n$ 时，需要用不可约多项式构造扩域，元素可以写成多项式的形式。
例子：构造 $\mathbb{F}_9$（$p=3, n=2$）。 基域 $\mathbb{F}_3 = \{0,1,2\}$。找一个不可约二次多项式，如 $x^2 + 1$（验证：$0^2+1=1$，$1^2+1=2$，$2^2+1=5 \equiv 2$，都不为 $0$，所以无根，不可约）。令 $i$ 满足 $i^2 + 1 = 0$，即 $i^2 = -1 = 2$。则 $\mathbb{F}_9 = \{a + bi : a,b \in \mathbb{F}_3\}$，共 9 个元素：$$\mathbb{F}_9 = \{0,\ 1,\ 2,\ i,\ 1{+}i,\ 2{+}i,\ 2i,\ 1{+}2i,\ 2{+}2i\}$$乘法例子：$(1+i)(2+i) = 2 + i + 2i + i^2 = 2 + 3i + 2 = 4 + 0 \cdot i = 1$。
为什么只能是素数幂？ ① 域中 $1+1+\cdots+1=0$ 的最小次数叫"特征"，必须是素数 $p$（否则能分解成 $a \cdot b = 0$，违反域无零因子的性质）；② 于是域必含子域 $\mathbb{F}_p$；③ 整个域可看作 $\mathbb{F}_p$ 上的 $n$ 维向量空间，每个坐标有 $p$ 种取值，故总共 $p^n$ 个元素。

什么是代数闭包 $\bar{K}$？ 直观地说，代数闭包是"把所有多项式的根都加进来"后得到的最小的域。
• $\mathbb{R}$（实数）的代数闭包是 $\mathbb{C}$（复数），因为 $x^2 + 1 = 0$ 在 $\mathbb{R}$ 中无解，但在 $\mathbb{C}$ 中有解 $\pm i$。
• $\mathbb{Q}$（有理数）的代数闭包是 $\bar{\mathbb{Q}}$（代数数），注意这不是 $\mathbb{C}$！因为 $\pi$ 和 $e$ 是超越数，不是任何有理系数多项式的根。
• $\mathbb{F}_q$（有限域）的代数闭包 $\bar{\mathbb{F}}_q$ 是所有 $\mathbb{F}_{q^n}$（$n = 1, 2, 3, \ldots$）的并集，是一个无限域。
为什么？ 代数闭包要让每个多项式都有根。$\mathbb{F}_q$ 上的 $n$ 次不可约多项式的根在 $\mathbb{F}_{q^n}$ 中。由于任意次数的不可约多项式都存在，就需要所有的 $\mathbb{F}_{q^n}$；而 $n$ 可以无限大，所以并集是无限域。
$$E: y^2 + a_1 xy + a_3 y = x^3 + a_2 x^2 + a_4 x + a_6, \tag{2.1}$$

where $a_1, \ldots, a_6 \in K$. Equation (2.1) is called the general Weierstrass equation for elliptic curves. Aside from all the $(x, y) \in \bar{K}$ solutions to the equation above, there is one extra point which can not be defined using the affine equation, but which must be included to complete the group definition. This point is called the point at infinity, which we denote by $\mathcal{O}$, and we will define it properly in a moment.

其中 $a_1, \ldots, a_6 \in K$。方程 (2.1) 叫做椭圆曲线的"一般 Weierstrass 方程"。除了上述方程的所有 $(x, y) \in \bar{K}$ 解之外，还有一个额外的点无法用仿射方程定义，但必须包含进来才能完成群的定义。这个点叫做"无穷远点"，记作 $\mathcal{O}$，我们稍后会正式定义它。

If $a_1, \ldots, a_6 \in K$, then we say $E$ is defined over $K$, and write this as $E/K$ (the same goes for any extension field $L$ of $K$). Before we go any further, we make a convenient simplification of the general Weierstrass equation. If the field characteristic is not 2 or 3, then divisions by 2 and 3 in $K$ permit the substitutions $y \mapsto (y - a_1 x - a_3)/2$ to give $E: y^2 = 4x^3 + b_2 x^2 + 2b_4 x + b_6$, and then $(x, y) \mapsto \left(\frac{x - 3b_2}{36}, \frac{y}{108}\right)$, which (upon appropriate rescaling) yields the following simplified equation.

如果 $a_1, \ldots, a_6 \in K$，我们就说 $E$ "定义在 $K$ 上"，写作 $E/K$（对于 $K$ 的任意扩域 $L$ 也一样）。在继续之前，我们对一般 Weierstrass 方程做一个方便的简化。如果域的特征不是 2 或 3，那么在 $K$ 中可以除以 2 和 3，允许做变量替换 $y \mapsto (y - a_1 x - a_3)/2$ 得到 $E: y^2 = 4x^3 + b_2 x^2 + 2b_4 x + b_6$，然后 $(x, y) \mapsto \left(\frac{x - 3b_2}{36}, \frac{y}{108}\right)$，经过适当的缩放后得到如下简化方程：
推导过程：从一般形式到短形式

起点：一般 Weierstrass 方程 $$y^2 + a_1 xy + a_3 y = x^3 + a_2 x^2 + a_4 x + a_6$$ 第一步：替换 $y \mapsto \frac{1}{2}(y - a_1 x - a_3)$，即 $y_{\text{old}} = \frac{1}{2}(y_{\text{new}} - a_1 x - a_3)$
代入左边：$y_{\text{old}}^2 + a_1 x y_{\text{old}} + a_3 y_{\text{old}}$
$= \frac{1}{4}(y - a_1 x - a_3)^2 + \frac{a_1 x}{2}(y - a_1 x - a_3) + \frac{a_3}{2}(y - a_1 x - a_3)$
展开并整理后，交叉项消掉，得到 $\frac{1}{4}y^2$（乘以 4 后）： $$y^2 = 4x^3 + b_2 x^2 + 2b_4 x + b_6$$ 其中 $b_2 = a_1^2 + 4a_2$，$b_4 = a_1 a_3 + 2a_4$，$b_6 = a_3^2 + 4a_6$。

第二步：替换 $(x, y) \mapsto \left(\frac{x - 3b_2}{36}, \frac{y}{108}\right)$，即 $x_{\text{old}} = \frac{x_{\text{new}} - 3b_2}{36}$
这个替换消去 $x^2$ 项。代入并整理（过程略），最终得到： $$y^2 = x^3 + ax + b$$ 其中 $a, b$ 是由原始系数 $a_1, \ldots, a_6$ 计算出的新系数。

关键点：这些替换需要除以 2 和 3，所以要求域的特征不是 2 或 3（否则除法无定义）。
$$E: y^2 = x^3 + ax + b. \tag{2.2}$$

Equation (2.2) is called the short Weierstrass equation for elliptic curves, and will be used all the way through this text. Namely, we will always be working over large prime fields, where the short Weierstrass equation covers all possible isomorphism classes of elliptic curves, so the curves we use will always be an instance of (2.2).

方程 (2.2) 叫做椭圆曲线的"短 Weierstrass 方程"，全文都将使用这个形式。也就是说，我们总是在大素数域上工作，短 Weierstrass 方程能覆盖所有可能的椭圆曲线同构类，所以我们用的曲线都是 (2.2) 的实例。

Example 2.0.1 (Magma script). $E/\mathbb{Q}: y^2 = x^3 - 2$ is an elliptic curve. Along with the point at infinity $\mathcal{O}$ (which we are still yet to define), the set of points over $\mathbb{Q}$ is written as $E(\mathbb{Q})$, and is defined as $E(\mathbb{Q}) = \{(x, y) \in \mathbb{A}^2(\mathbb{Q}): y^2 = x^3 - 2\} \cup \{\mathcal{O}\}$. The point $P = (x_P, y_P) = (3, 5)$ lies in $E(\mathbb{Q})$, as do $Q = (x_Q, y_Q) = \left(\frac{129}{100}, -\frac{383}{1000}\right)$ and $R = (x_R, y_R) = \left(\frac{164323}{29241}, -\frac{66234835}{5000211}\right)$, so we can write $P, Q, R \in E(\mathbb{Q})$. We usually write $E$ to represent the group of points over the full algebraic closure, so for example, the point $S = (x_S, y_S) = \left(0, \sqrt{-2}\right) \in E = E(\bar{\mathbb{Q}})$, but $S \not\in E(\mathbb{Q})$.

Soon we will be defining the binary group operation $\oplus$ on $E$ using rational formulas in the underlying field, so an active reader can return to this example with these formulas to verify that $R = P \oplus Q$, where $x_R, y_R$ are computed from $x_P, y_P, x_Q, y_Q$ using additions and multiplications (also subtractions and inversions) in $\mathbb{Q}$. Furthermore, it can also be verified that $Q = P \oplus P$, so that $R = P \oplus P \oplus P$; we usually write these as $Q = [2]P$ and $R = [3]P$, where $\underbrace{P \oplus P \cdots \oplus P}_{n} = [n]P$ in general. To finish this example, we remark that if $(x', y') \in E$, then $(x', -y') \in E$ (but is not distinct if $y' = 0$), which is true for any elliptic curve in short Weierstrass form.

$E/\mathbb{Q}: y^2 = x^3 - 2$ 是一条椭圆曲线。连同无穷远点 $\mathcal{O}$（我们还没定义），$\mathbb{Q}$ 上的点集记作 $E(\mathbb{Q})$。点 $P = (3, 5)$、$Q = \left(\frac{129}{100}, -\frac{383}{1000}\right)$ 和 $R = \left(\frac{164323}{29241}, -\frac{66234835}{5000211}\right)$ 都在 $E(\mathbb{Q})$ 上。我们很快会定义群运算 $\oplus$，读者可以验证 $R = P \oplus Q$，且 $Q = [2]P$，$R = [3]P$。另外，如果 $(x', y') \in E$，那么 $(x', -y') \in E$（除非 $y' = 0$）。

Example 2.0.2 (Magma script). $E/\mathbb{F}_{11}: y^2 = x^3 + 4x + 3$ is an elliptic curve. $E(\mathbb{F}_{11})$ has 14 points: $(0, 5), (0, 6), (3, 3), (3, 8), (5, 4), (5, 7), (6, 1), (6, 10), (7, 0), (9, 3), (9, 8), (10, 3), (10, 8)$, not forgetting the point at infinity $\mathcal{O}$. Notice that all but two points come in pairs $(x', y')$ and $(x', -y')$, the exceptions being $(x', y') = (7, 0)$ (since $y' = -y' = 0$) and $\mathcal{O}$. If we form the quadratic extension $\mathbb{F}_{q^2} = \mathbb{F}_q(i)$ with $i^2 + 1 = 0$, then considering $E$ over $\mathbb{F}_{q^2}$ will allow many more solutions, and give many more points: namely, $\#E(\mathbb{F}_{q^2}) = 140$. In addition to the points in $E(\mathbb{F}_q)$, $E(\mathbb{F}_{q^2})$ will also contain those points with $x$-coordinates in $\mathbb{F}_q$ that did not give $x^3 + 4x + 3$ as a quadratic residue in $\mathbb{F}_q$ (but necessarily do in $\mathbb{F}_{q^2}$), and many more with both coordinates in $\mathbb{F}_{q^2} \setminus \mathbb{F}_q$. Examples of both such points are $(2, 5i)$ and $(2i + 10, 7i + 2)$ respectively. It is not a coincidence that $\#E(\mathbb{F}_q) \mid \#E(\mathbb{F}_{q^2})$, since $E(\mathbb{F}_q)$ is a subgroup of $E(\mathbb{F}_{q^2})$.

$E/\mathbb{F}_{11}: y^2 = x^3 + 4x + 3$ 是一条椭圆曲线。$E(\mathbb{F}_{11})$ 有 14 个点（别忘了无穷远点 $\mathcal{O}$）。除了两个例外，所有点都成对出现 $(x', y')$ 和 $(x', -y')$。例外是 $(7, 0)$（因为 $y' = -y' = 0$）和 $\mathcal{O}$。如果构造二次扩域 $\mathbb{F}_{q^2} = \mathbb{F}_q(i)$（$i^2 + 1 = 0$），在 $\mathbb{F}_{q^2}$ 上考虑 $E$ 会有更多点：$\#E(\mathbb{F}_{q^2}) = 140$。$\#E(\mathbb{F}_q) \mid \#E(\mathbb{F}_{q^2})$ 不是巧合，因为 $E(\mathbb{F}_q)$ 是 $E(\mathbb{F}_{q^2})$ 的子群。

Not every tuple $(a, b) \in K \times K$ gives rise to the curve given by $f(x, y) = y^2 - (x^3 + ax + b) = 0$ being an elliptic curve. If there exists $P = (x_P, y_P)$ on $f$ such that both partial derivatives $\frac{\partial f}{\partial x}$ and $\frac{\partial f}{\partial y}$ vanish simultaneously at $P$, then $P$ is called a singular point and $f$ is also deemed singular. Conversely, if no such point exists, $f$ is called non-singular, or smooth, and is then an elliptic curve. It is easy enough to show that a singularity occurs if and only if $4a^3 + 27b^2 = 0$ (see [Sil09, Ch. III.1, Prop. 1.4]), so as long as $4a^3 + 27b^2 \neq 0$ in $K$, then $E/K: y^2 = x^3 + ax + b$ is an elliptic curve.

不是每个元组 $(a, b) \in K \times K$ 都能使 $f(x, y) = y^2 - (x^3 + ax + b) = 0$ 成为椭圆曲线。如果存在 $f$ 上的点 $P = (x_P, y_P)$ 使得两个偏导数 $\frac{\partial f}{\partial x}$ 和 $\frac{\partial f}{\partial y}$ 在 $P$ 点同时为零，那么 $P$ 叫做奇异点，$f$ 也被称为奇异的。反之，如果不存在这样的点，$f$ 就叫做非奇异的或光滑的，这时它才是椭圆曲线。容易证明，奇点存在当且仅当 $4a^3 + 27b^2 = 0$，所以只要在 $K$ 中 $4a^3 + 27b^2 \neq 0$，$E/K: y^2 = x^3 + ax + b$ 就是椭圆曲线。
