3.4 The correspondence $V$

3.4 对应 $V$

$k$ is any field, and $A = k[X_1, \ldots, X_n]$. Following an almost universal idiosyncrasy/ˌɪd.i.əˈsɪŋ.krə.si/ 特殊习惯 of algebraic geometers, I write $\mathbb{A}^n_k = k^n$ for the $n$-dimensional affine/əˈfaɪn/ 仿射的 space over $k$; given a polynomial $f(X_1, \ldots, X_n) \in A$ and a point $P = (a_1, \ldots, a_n) \in \mathbb{A}^n_k$, the element $f(a_1, \ldots, a_n) \in k$ is thought of as 'evaluating the function $f$ at $P$'. Define a correspondence/ˌkɒr.ɪˈspɒn.dəns/ 对应

$$\{\text{ideals } J \subset A\} \longrightarrow \{\text{subsets } X \subset \mathbb{A}^n_k\}$$

by

$$J \longmapsto V(J) = \{P \in \mathbb{A}^n_k \mid f(P) = 0 \;\forall f \in J\}.$$
$k$ 是任意域，$A = k[X_1,\ldots,X_n]$。代数几何学家有一个几乎普遍的特殊记法：用 $\mathbb{A}^n_k = k^n$ 表示 $k$ 上的 $n$ 维仿射空间。给定多项式 $f(X_1,\ldots,X_n) \in A$ 和点 $P=(a_1,\ldots,a_n) \in \mathbb{A}^n_k$，$f(a_1,\ldots,a_n) \in k$ 被看作"在 $P$ 点处求值"。

定义从理想到子集的对应：$J \mapsto V(J) = \{P \in \mathbb{A}^n_k \mid \text{对所有 } f \in J,\; f(P)=0\}$。

Definition   A subset $X \subset \mathbb{A}^n_k$ is an algebraic set if $X = V(I)$ for some $I$. (This is the same thing as a variety, but I want to reserve/rɪˈzɜːrv/ 保留 the word.) Notice that by Corollary 3.3, $I$ is finitely generated. If $I = (f_1, \ldots, f_r)$ then clearly

$$V(I) = \{P \in \mathbb{A}^n_k \mid f_i(P) = 0 \text{ for } i = 1, \ldots, r\},$$

so that an algebraic set is just a locus/ˈloʊ.kəs/ 轨迹，点集 of points satisfying a finite number of polynomial equations.

If $I = (f)$ is a principal/ˈprɪn.sɪ.pəl/ 主的（主理想） ideal, then I usually write $V(f)$ for $V(I)$; this is of course the same thing as $V : (f = 0)$ in the notation of Chapters 1--2.

定义：子集 $X \subset \mathbb{A}^n_k$ 是一个代数集，如果 $X = V(I)$ 对某个理想 $I$ 成立。（本质上就是"簇"，但这个词我先留着后面用。）注意根据推论3.3，$I$ 是有限生成的。如果 $I=(f_1,\ldots,f_r)$，那么 $$V(I) = \{P \in \mathbb{A}^n_k \mid f_i(P) = 0,\; i=1,\ldots,r\},$$ 所以代数集就是满足有限个多项式方程的点的集合。

如果 $I=(f)$ 是主理想，通常简写 $V(f)$ 代替 $V(I)$，这和第1--2章中的记号 $V:(f=0)$ 是一回事。
