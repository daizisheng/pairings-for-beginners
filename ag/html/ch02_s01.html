2.1 Examples of parametrised cubics

2.1 参数化三次曲线的例子

Some plane cubic curves can be parametrised/ˈpær.ə.mɪ.traɪzd/ 参数化的, just as the conics:

有些平面三次曲线可以被参数化，就像圆锥曲线一样：

Nodal cubic   $C : (y^2 = x^3 + x^2) \subset \mathbb{R}^2$ is the image of the map $\varphi\colon \mathbb{R}^1 \to \mathbb{R}^2$ given by $t \mapsto (t^2 - 1,\, t^3 - t)$ (check it and see);

结点三次曲线   $C : (y^2 = x^3 + x^2) \subset \mathbb{R}^2$ 是映射 $\varphi\colon \mathbb{R}^1 \to \mathbb{R}^2$（由 $t \mapsto (t^2 - 1,\, t^3 - t)$ 给出）的像（自己验证一下）；

Cuspidal/ˈkʌs.pɪ.dəl/ 尖点的 cubic   $C : (y^2 = x^3) \subset \mathbb{R}^2$ is the image of $\varphi\colon \mathbb{R}^1 \to \mathbb{R}^2$ given by $t \mapsto (t^2,\, t^3)$:

尖点三次曲线   $C : (y^2 = x^3) \subset \mathbb{R}^2$ 是映射 $\varphi\colon \mathbb{R}^1 \to \mathbb{R}^2$（由 $t \mapsto (t^2,\, t^3)$ 给出）的像：

Think about the singularities/ˌsɪŋ.ɡjʊˈlær.ɪ.tiz/ 奇点 of the image curve, and of the map $\varphi$. These examples will occur throughout the course, so spend some time playing with the equations; see Ex. 2.1–2.

想一想像曲线的奇点，以及映射 $\varphi$ 的奇点。这些例子会贯穿整门课程，所以花点时间好好玩一下这些方程；参见习题 2.1–2。
