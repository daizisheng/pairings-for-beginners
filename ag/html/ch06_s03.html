6.3 Proposition: $V_{\mathrm{nonsing}}$ is dense

6.3 命题：$V_{\mathrm{nonsing}}$ 稠密

Proposition 6.3   $V_{\mathrm{nonsing}} = \{P \in V \mid P \text{ is nonsingular}\}$ is a dense/dens/ 稠密的 open set of $V$ for the Zariski topology.

命题 6.3：$V_{\mathrm{nonsing}} = \{P \in V \mid P \text{ 是非奇异点}\}$ 在 Zariski 拓扑下是 $V$ 的一个稠密开集。

Proof   The complement/ˈkɒm.plɪ.ment/ 补集 of $V_{\mathrm{nonsing}}$ is the set $V_{\mathrm{sing}}$ of singular points, which is defined by $\partial f / \partial X_i(P) = 0$ for all $i$, that is

$$V_{\mathrm{sing}} = V\!\left(f, \frac{\partial f}{\partial X_1}, \ldots, \frac{\partial f}{\partial X_n}\right) \subset \mathbb{A}^n,$$

which is closed by definition of the Zariski topology. Since $V$ is irreducible (by (3.11, a)), to show that the open $V_{\mathrm{nonsing}}$ is dense, I only have to show it's nonempty (by Proposition 4.2); arguing by contradiction/ˌkɒn.trəˈdɪk.ʃən/ 矛盾, suppose that it's empty, that is, suppose $V = V(f) = V_{\mathrm{sing}}$. Then each of the polynomials $\partial f / \partial X_i$ must vanish on $V$, therefore (by (3.11) once more) they must be divisible/dɪˈvɪz.ɪ.bl/ 可整除的 by $f$ in $k[X_1, \ldots, X_n]$; but viewed as a polynomial in $X_i$, $\partial f / \partial X_i$ has degree strictly smaller than $f$, so that $f$ divides $\partial f / \partial X_i$ implies that in fact $\partial f / \partial X_i = 0$ as a polynomial.

证明：$V_{\mathrm{nonsing}}$ 的补集是奇异点集 $V_{\mathrm{sing}}$，它由 $\partial f / \partial X_i(P) = 0$（对所有 $i$）所定义，即 $$V_{\mathrm{sing}} = V\!\left(f, \frac{\partial f}{\partial X_1}, \ldots, \frac{\partial f}{\partial X_n}\right) \subset \mathbb{A}^n,$$ 由 Zariski 拓扑的定义知这是闭集。由于 $V$ 不可约（由(3.11, a)），要证明开集 $V_{\mathrm{nonsing}}$ 稠密，只需证明它非空（由命题4.2）；用反证法，假设它为空，即假设 $V = V(f) = V_{\mathrm{sing}}$。则每个多项式 $\partial f / \partial X_i$ 都必须在 $V$ 上为零，因此（再由(3.11)）它们在 $k[X_1, \ldots, X_n]$ 中必须被 $f$ 整除；但把 $\partial f / \partial X_i$ 看作关于 $X_i$ 的多项式，它的次数严格小于 $f$，所以 $f$ 整除 $\partial f / \partial X_i$ 意味着实际上 $\partial f / \partial X_i = 0$（作为多项式）。

Over $\mathbb{C}$, this is obviously only possible if $X_i$ does not appear in $f$, and if this happens for all $i$ then $f = \text{const.} \in \mathbb{C}$, which is excluded. Over a general field $k$, $\partial f / \partial X_i = 0$ is only possible if $f$ is an inseparable/ɪnˈsep.ər.ə.bl/ 不可分的 polynomial in $X_i$, that is, $\operatorname{char} k = p$, and $X_i$ only appears in $f$ as the $p$th power $X_i^p$. If this happens for each $i$, then by the argument given in (3.16), $f$ is a $p$th power in $k[X_1, \ldots, X_n]$; this contradicts the fact that $f$ is irreducible.   Q.E.D.

在 $\mathbb{C}$ 上，这显然只有当 $X_i$ 不出现在 $f$ 中时才可能，如果所有 $i$ 都如此，则 $f = \text{常数} \in \mathbb{C}$，这与假设矛盾。在一般域 $k$ 上，$\partial f / \partial X_i = 0$ 只有当 $f$ 是关于 $X_i$ 的不可分多项式时才可能，即 $\operatorname{char} k = p$，且 $X_i$ 在 $f$ 中只以 $p$ 次幂 $X_i^p$ 的形式出现。如果每个 $i$ 都是这样，则由(3.16)中给出的论证，$f$ 在 $k[X_1, \ldots, X_n]$ 中是一个 $p$ 次幂；这与 $f$ 不可约矛盾。Q.E.D.
