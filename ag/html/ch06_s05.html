6.5 Proposition: $\dim T_P V$ is upper semicontinuous

6.5 命题：$\dim T_P V$ 上半连续

Proposition 6.5   The function $V \to \mathbb{N}$ defined by $P \mapsto \dim T_P V$ is an upper semicontinuous/ˌʌp.ə ˌsem.i.kənˈtɪn.ju.əs/ 上半连续的 function (in the Zariski topology of $V$). In other words, for any integer $r$, the subset

$$S(r) = \{ P \in V \mid \dim T_P V \geq r \} \subset V$$

is closed.

命题 6.5：由 $P \mapsto \dim T_P V$ 定义的函数 $V \to \mathbb{N}$ 是上半连续函数（在 $V$ 的 Zariski 拓扑下）。换言之，对任意整数 $r$，子集 $$S(r) = \{ P \in V \mid \dim T_P V \geq r \} \subset V$$ 是闭集。

Proof   Let $(f_1, \ldots, f_m)$ be a set of generators/ˈdʒen.ə.reɪ.tərz/ 生成元 of $I(V)$; it is easy to see that for any $g \in I(V)$, the linear part $g_P^{(1)}$ of $g$ is a linear combination of those of the $f_i$, so that the definition of $T_P V$ simplifies to

$$T_P V = \bigcap_{i=1}^m \left( f_{i,P}^{(1)} = 0 \right) \subset \mathbb{A}^n.$$

Then by elementary linear algebra,

$$P \in S(r) \iff \text{the matrix } \left( \frac{\partial f_i}{\partial X_j}(P) \right)_{\substack{i=1,\ldots,m \\ j=1,\ldots,n}} \text{ has rank} \leq n - r$$ $$\iff \text{every } (n - r + 1) \times (n - r + 1) \text{ minor vanishes.}$$

Now each entry $\partial f_i / \partial X_j(P)$ of the matrix is a polynomial function of $P$; thus each minor/ˈmaɪ.nər/ 子式 is a determinant/dɪˈtɜː.mɪ.nənt/ 行列式 of a matrix of polynomials, and so is itself a polynomial. Hence $S(r) \subset V \subset \mathbb{A}^n$ is an algebraic subset.   Q.E.D.

证明：设 $(f_1, \ldots, f_m)$ 是 $I(V)$ 的一组生成元；容易看出，对任意 $g \in I(V)$，$g$ 的线性部分 $g_P^{(1)}$ 是各 $f_i$ 的线性部分的线性组合，因此 $T_P V$ 的定义简化为 $$T_P V = \bigcap_{i=1}^m \left( f_{i,P}^{(1)} = 0 \right) \subset \mathbb{A}^n.$$ 然后由初等线性代数， $$P \in S(r) \iff \text{矩阵 } \left( \frac{\partial f_i}{\partial X_j}(P) \right)_{\substack{i=1,\ldots,m \\ j=1,\ldots,n}} \text{ 的秩} \leq n - r$$ $$\iff \text{所有 } (n - r + 1) \times (n - r + 1) \text{ 阶子式为零。}$$ 矩阵的每个元素 $\partial f_i / \partial X_j(P)$ 都是 $P$ 的多项式函数；因此每个子式是多项式矩阵的行列式，本身也是多项式。所以 $S(r) \subset V \subset \mathbb{A}^n$ 是代数子集。Q.E.D.
