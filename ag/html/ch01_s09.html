1.9 Easy cases of Bézout's Theorem

Bézout 定理的简单情形

Bézout's theorem says that if $C$ and $D$ are plane curves of degrees $\deg C = m$, $\deg D = n$, then the number of points of intersection/ˌɪn.təˈsek.ʃən/ 交点，交集 of $C$ and $D$ is $mn$, provided that (i) the field is algebraically closed; (ii) points of intersection are counted with the right multiplicities/ˌmʌl.tɪˈplɪs.ɪ.tiz/ 重数; (iii) we work in $\mathbb{P}^2$ to take right account of intersections 'at infinity'. See for example [Fulton, p. 112] for a self-contained proof. In this section I am going to treat the case when one of the curves is a line or conic.

Bézout 定理说的是：如果 $C$ 和 $D$ 是次数分别为 $\deg C = m$、$\deg D = n$ 的平面曲线，那么 $C$ 与 $D$ 的交点个数为 $mn$，前提是 (i) 基域是代数封闭的；(ii) 交点按正确的重数计数；(iii) 在射影平面 $\mathbb{P}^2$ 中工作，以正确处理"无穷远处"的交点。完整证明见 [Fulton, p. 112]。本节只讨论其中一条曲线是直线或二次曲线的情形。

Theorem Let $L \subset \mathbb{P}^2_k$ be a line (respectively $C \subset \mathbb{P}^2_k$ a nondegenerate/ˌnɒn.dɪˈdʒen.ər.ɪt/ 非退化的 conic), and let $D \subset \mathbb{P}^2_k$ be a curve defined by $D : (G_d(X, Y, Z) = 0)$, where $G$ is a form of degree $d$ in $X, Y, Z$. Assume that $L \not\subset D$ (respectively, $C \not\subset D$); then

$$\#\{L \cap D\} \leq d \quad (\text{respectively } \#\{C \cap D\} \leq 2d).$$

In fact there is a natural definition of multiplicity of intersection such that the inequality/ˌɪn.ɪˈkwɒl.ɪ.ti/ 不等式 still holds for 'number of points counted with multiplicities', and if $k$ is algebraically closed then equality holds.

定理 设 $L \subset \mathbb{P}^2_k$ 是一条直线（对应地，$C \subset \mathbb{P}^2_k$ 是一条非退化二次曲线），$D \subset \mathbb{P}^2_k$ 是由 $D: (G_d(X,Y,Z) = 0)$ 定义的曲线，其中 $G$ 是 $X,Y,Z$ 的 $d$ 次齐次式。假设 $L \not\subset D$（对应地 $C \not\subset D$），则 $$\#\{L \cap D\} \leq d \quad (\text{对应地 } \#\{C \cap D\} \leq 2d).$$ 事实上，存在一种自然的交点重数定义，使得上述不等式对"按重数计数的交点个数"仍然成立；若 $k$ 是代数封闭域，则等号成立。

Proof A line $L \subset \mathbb{P}^2_k$ is given by an equation $\lambda = 0$, with $\lambda$ a linear form; for my purpose, it is convenient to give it parametrically/ˌpær.əˈmet.rɪ.kəl.i/ 参数化地 as

$$X = a(U, V), \quad Y = b(U, V), \quad Z = c(U, V),$$

where $a, b, c$ are linear forms in $U, V$. So for example, if $\lambda = \alpha X + \beta Y + \gamma Z$, and $\gamma \neq 0$, then $L$ can be given as

$$X = U, \quad Y = V, \quad Z = -\frac{\alpha}{\gamma}U - \frac{\beta}{\gamma}V.$$

Similarly, as explained in (1.7), a nondegenerate conic can be given parametrically as

$$X = a(U, V), \quad Y = b(U, V), \quad Z = c(U, V),$$

where $a, b, c$ are quadratic/kwɒˈdræt.ɪk/ 二次的 forms in $U, V$. This is because $C$ is a projective transformation of $(XZ = Y^2)$, which is parametrically $(X, Y, Z) = (U^2, UV, V^2)$, so $C$ is given by

$$\begin{pmatrix} X \\ Y \\ Z \end{pmatrix} = M \begin{pmatrix} U^2 \\ UV \\ V^2 \end{pmatrix}$$

where $M$ is a nonsingular $3 \times 3$ matrix.

证明 直线 $L \subset \mathbb{P}^2_k$ 由方程 $\lambda = 0$ 给出，其中 $\lambda$ 是线性形式；为了当前目的，用参数形式表示更方便：$X = a(U,V)$，$Y = b(U,V)$，$Z = c(U,V)$，其中 $a,b,c$ 是 $U,V$ 的线性形式。例如，如果 $\lambda = \alpha X + \beta Y + \gamma Z$ 且 $\gamma \neq 0$，则 $L$ 可以表示为 $X = U$，$Y = V$，$Z = -\frac{\alpha}{\gamma}U - \frac{\beta}{\gamma}V$。类似地，如 (1.7) 所述，非退化二次曲线可以参数化为 $X = a(U,V)$，$Y = b(U,V)$，$Z = c(U,V)$，其中 $a,b,c$ 是 $U,V$ 的二次形式。这是因为 $C$ 是 $(XZ = Y^2)$ 的射影变换，而 $(XZ = Y^2)$ 的参数形式为 $(X,Y,Z) = (U^2, UV, V^2)$，所以 $C$ 由 $$\begin{pmatrix} X \\ Y \\ Z \end{pmatrix} = M \begin{pmatrix} U^2 \\ UV \\ V^2 \end{pmatrix}$$ 给出，其中 $M$ 是非奇异的 $3 \times 3$ 矩阵。

Then the intersection of $L$ (respectively $C$) with $D$ is given by finding the values of the ratios $(U : V)$ such that

$$F(U, V) = G_d(a(U, V), b(U, V), c(U, V)) = 0.$$

But $F$ is a form of degree $d$ (respectively $2d$) in $U, V$, so the result follows by (1.8). Q.E.D.

于是 $L$（对应地 $C$）与 $D$ 的交点就是使得 $$F(U, V) = G_d(a(U, V), b(U, V), c(U, V)) = 0$$ 成立的比值 $(U : V)$。而 $F$ 是 $U,V$ 的 $d$ 次（对应地 $2d$ 次）齐次式，由 (1.8) 即得结论。证毕。
