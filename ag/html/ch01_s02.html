1.2 Similar example

1.2 类似的例子

$C : (2X^2 + Y^2 = 5Z^2)$. The same method leads to the parametrisation/ˌpær.ə.mɪ.traɪˈzeɪ.ʃən/ 参数化 $\mathbb{R} \to C$ given by

$$x = \frac{2\sqrt{5}\,\lambda}{1 + 2\lambda^2}, \quad y = \frac{2\lambda^2 - 1}{1 + 2\lambda^2}.$$

This allows us to understand all about points of $C$ with coefficients/ˌkoʊ.ɪˈfɪʃ.ənts/ 系数 in $\mathbb{R}$, and there's no real difference from the previous example; what about $\mathbb{Q}$?

考虑曲线 $C: (2X^2 + Y^2 = 5Z^2)$。用和上一节同样的方法可以得到参数化 $\mathbb{R} \to C$： $$x = \frac{2\sqrt{5}\,\lambda}{1 + 2\lambda^2}, \quad y = \frac{2\lambda^2 - 1}{1 + 2\lambda^2}.$$ 这让我们完全理解了 $C$ 上系数在 $\mathbb{R}$ 中的点，与上一个例子没有本质区别。那有理数 $\mathbb{Q}$ 上的情况呢？

Proposition   If $(a, b, c) \in \mathbb{Q}$ satisfies $2a^2 + b^2 = 5c^2$ then $(a, b, c) = (0, 0, 0)$.

命题：如果 $(a,b,c) \in \mathbb{Q}$ 满足 $2a^2 + b^2 = 5c^2$，那么 $(a,b,c) = (0,0,0)$。

Proof   Multiplying through by a common denominator/dɪˈnɒm.ɪ.neɪ.tər/ 分母 and taking out a common factor if necessary, I can assume that $a, b, c$ are integers, not all of which are divisible/dɪˈvɪz.ə.bəl/ 可整除的 by 5; also if $5 \mid a$ and $5 \mid b$ then $25 \mid 5c^2$, so that $5 \mid c$, which contradicts/ˌkɒn.trəˈdɪkts/ 与...矛盾 what I've just said. It is now easy to get a contradiction by considering the possible values of $a$ and $b$ mod 5: since any square is $0$, $1$ or $4$ mod 5, clearly $2a^2 + b^2$ is one of $0+1$, $0+4$, $2+0$, $2+1$, $2+4$, $8+0$, $8+1$ or $8+4$ mod 5, none of which can be of the form $5c^2$.   Q.E.D.

证明：先乘以公分母，再约去公因子，可以假设 $a,b,c$ 都是整数，而且不全被5整除。另外，如果 $5\mid a$ 且 $5\mid b$，那么 $25\mid 5c^2$，所以 $5\mid c$，这就和"不全被5整除"矛盾了。现在考虑 $a$ 和 $b$ 模5的所有可能值就容易得到矛盾：任何平方数模5只能是 $0$、$1$ 或 $4$，所以 $2a^2 + b^2$ 模5只能取 $0+1$、$0+4$、$2+0$、$2+1$、$2+4$、$8+0$、$8+1$ 或 $8+4$ 这些值，没有一个能被写成 $5c^2$ 的形式。证毕。

Note that this is a thoroughly/ˈθʌr.ə.li/ 彻底地 arithmetic/ə.rɪθˈmet.ɪk/ 算术的 argument.

注意这是一个纯粹的算术论证。
