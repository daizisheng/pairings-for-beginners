4.5 Corollary: $f^*$ is an Isomorphism

推论：$f^*$ 是同构

Corollary 4.5 A polynomial map $f \colon V \to W$ is an isomorphism if and only if $f^* \colon k[W] \to k[V]$ is an isomorphism.

推论 4.5 多项式映射 $f \colon V \to W$ 是同构，当且仅当 $f^* \colon k[W] \to k[V]$ 是同构。

Example Over an infinite field $k$, the polynomial map

$$\varphi \colon \mathbb{A}^1_k \to C \colon (Y^2 = X^3) \subset \mathbb{A}^2_k \quad \text{given by} \quad T \mapsto (T^2, T^3)$$

is not an isomorphism. For in this case, the homomorphism

$$\varphi^* \colon k[C] = k[X, Y]/(Y^2 - X^3) \to k[T]$$

is given by $X \mapsto T^2$, $Y \mapsto T^3$. The image of $\varphi^*$ is the $k$-algebra generated by $T^2, T^3$, that is $k[T^2, T^3] \subsetneq k[T]$. (Please make sure you understand why $T^2, T^3$ don't generate $k[T]$; I can't help you on this.)

例子 在无限域 $k$ 上，多项式映射 $$\varphi \colon \mathbb{A}^1_k \to C \colon (Y^2 = X^3) \subset \mathbb{A}^2_k, \quad T \mapsto (T^2, T^3)$$ 不是同构。因为在这种情况下，同态 $$\varphi^* \colon k[C] = k[X, Y]/(Y^2 - X^3) \to k[T]$$ 由 $X \mapsto T^2$，$Y \mapsto T^3$ 给出。$\varphi^*$ 的像是由 $T^2, T^3$ 生成的 $k$-代数，即 $k[T^2, T^3] \subsetneq k[T]$。（请确保你理解为什么 $T^2, T^3$ 不能生成 $k[T]$；这个我帮不了你。）

Notice that $\varphi$ is bijective/baɪˈdʒek.tɪv/ 双射的, and so has a perfectly good inverse map $\psi \colon C \to \mathbb{A}^1_k$ given by $(X, Y) \mapsto 0$ if $X = Y = 0$ and $Y/X$ otherwise. So why isn't $\varphi$ an isomorphism? The point is that $C$ has fewer polynomial functions on it than $\mathbb{A}^1$; in a sense you can see that for yourself, since $k[\mathbb{A}^1] = k[T]$ has a polynomial function with nonzero derivative/dɪˈrɪv.ə.tɪv/ 导数 at 0. The gut feeling is that $\varphi$ 'squashes up the tangent/ˈtæn.dʒənt/ 切线；切向量 vector at 0'.

注意 $\varphi$ 是双射的，因此有一个完全合理的逆映射 $\psi \colon C \to \mathbb{A}^1_k$，定义为：当 $X = Y = 0$ 时 $(X, Y) \mapsto 0$，否则 $(X, Y) \mapsto Y/X$。那为什么 $\varphi$ 不是同构呢？关键在于 $C$ 上的多项式函数比 $\mathbb{A}^1$ 上的少；你自己也能体会到这一点，因为 $k[\mathbb{A}^1] = k[T]$ 有一个在 0 处导数非零的多项式函数。直觉上的感觉是 $\varphi$ "在 0 处压缩了切向量"。
