1.3 Conics in $\mathbb{R}^2$

1.3 $\mathbb{R}^2$ 中的二次曲线

A conic/ˈkɒn.ɪk/ 二次曲线，圆锥曲线 in $\mathbb{R}^2$ is a plane curve given by a quadratic/kwɒˈdræt.ɪk/ 二次的 equation

$$q(x, y) = ax^2 + bxy + cy^2 + dx + ey + f = 0.$$

Everyone has seen the classification/ˌklæs.ɪ.fɪˈkeɪ.ʃən/ 分类 of nondegenerate/ˌnɒn.dɪˈdʒen.ər.ət/ 非退化的 conics:

$\mathbb{R}^2$ 中的二次曲线（圆锥曲线）就是由二次方程 $q(x,y) = ax^2 + bxy + cy^2 + dx + ey + f = 0$ 定义的平面曲线。大家都见过非退化二次曲线的分类：
图1.2：三种非退化的二次曲线：(a) 椭圆；(b) 抛物线；(c) 双曲线。

in addition, there are a number of peculiar/pɪˈkjuː.li.ər/ 特殊的，奇怪的 cases:

(d) single point given by $x^2 + y^2 = 0$;

(e, f, g) empty set given by any of the 3 equations: (e) $x^2 + y^2 = -1$, (f) $x^2 = -1$ or (g) $0 = 1$.

These three equations are different, although they define the same locus/ˈloʊ.kəs/ 轨迹，点集 of zeros in $\mathbb{R}^2$; consider for example their complex solutions.

(h) line $x = 0$;

(i) line pair $xy = 0$;

(j) parallel lines $x(x - 1) = 0$;

(k) 'double line' $x^2 = 0$; you can choose for yourself whether you'll allow the final case:

(l) whole plane given by $0 = 0$.

除此之外，还有一些特殊（退化）的情况：
(d) 单个点：$x^2 + y^2 = 0$；
(e, f, g) 空集，可以由三个不同的方程给出：(e) $x^2 + y^2 = -1$，(f) $x^2 = -1$，(g) $0 = 1$。这三个方程虽然不同，但在 $\mathbb{R}^2$ 中定义的零点集是一样的（都是空集）——不过你想想它们的复数解就知道区别了。
(h) 一条直线：$x = 0$；
(i) 两条相交直线（线对）：$xy = 0$；
(j) 两条平行线：$x(x-1) = 0$；
(k) "重线"（double line）：$x^2 = 0$；至于最后一种情况你自己决定要不要算：
(l) 整个平面：$0 = 0$。
