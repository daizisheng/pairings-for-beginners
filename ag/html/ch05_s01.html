5.1 Graded rings and homogeneous ideals

5.1 分次环与齐次理想

Definition   A polynomial/ˌpɒl.ɪˈnoʊ.mi.əl/ 多项式 $f \in k[X_0, \ldots, X_n]$ is homogeneous of degree $d$ if

$$f = \sum a_{i_0 \ldots i_n} X_0^{i_0} \cdots X_n^{i_n} \quad \text{with } a_{i_0 \ldots i_n} \neq 0 \text{ only if } i_0 + \cdots + i_n = d.$$

Any $f \in k[X_0, \ldots, X_n]$ has a unique expression $f = f_0 + f_1 + \cdots + f_N$ in which $f_d$ is homogeneous of degree $d$ for each $d = 0, 1, \ldots, N$.

定义   多项式 $f \in k[X_0, \ldots, X_n]$ 被称为 $d$ 次齐次的，如果 $$f = \sum a_{i_0 \ldots i_n} X_0^{i_0} \cdots X_n^{i_n}, \quad \text{且 } a_{i_0 \ldots i_n} \neq 0 \text{ 仅当 } i_0 + \cdots + i_n = d.$$ 任何 $f \in k[X_0, \ldots, X_n]$ 都有唯一的表达式 $f = f_0 + f_1 + \cdots + f_N$，其中每个 $f_d$ 是 $d$ 次齐次多项式（$d = 0, 1, \ldots, N$）。

Proposition   If $f$ is homogeneous of degree $d$ then

$$f(\lambda X_0, \ldots, \lambda X_n) = \lambda^d f(X_0, \ldots, X_n) \quad \text{for all } \lambda \in k;$$

if $k$ is an infinite/ˈɪn.fɪ.nɪt/ 无限的 field then the converse/ˈkɒn.vɜːs/ 逆命题 also holds.

命题   如果 $f$ 是 $d$ 次齐次的，则 $$f(\lambda X_0, \ldots, \lambda X_n) = \lambda^d f(X_0, \ldots, X_n), \quad \forall\, \lambda \in k;$$ 若 $k$ 是无限域，则逆命题也成立。

Proof   Try it and see.

证明   试试就知道了。

Definition   An ideal $I \subset k[X_0, \ldots, X_n]$ is homogeneous if for all $f \in I$, the homogeneous decomposition/ˌdiː.kɒm.pəˈzɪʃ.ən/ 分解 $f = f_0 + f_1 + \cdots + f_N$ of $f$ satisfies $f_i \in I$ for all $i$.

It is equivalent/ɪˈkwɪv.əl.ənt/ 等价的 to say that $I$ is generated by (finitely many) homogeneous polynomials.

定义   理想 $I \subset k[X_0, \ldots, X_n]$ 是齐次的，如果对所有 $f \in I$，$f$ 的齐次分解 $f = f_0 + f_1 + \cdots + f_N$ 满足每个 $f_i \in I$。

等价地说，$I$ 由（有限多个）齐次多项式生成。
