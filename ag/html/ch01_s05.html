1.5 Equation of a Conic

二次曲线的方程

The inhomogeneous/ˌɪn.hə.məˈdʒiː.ni.əs/ 非齐次的 quadratic polynomial

$$q(x, y) = ax^2 + bxy + cy^2 + dx + ey + f$$

corresponds to the homogeneous quadratic

$$Q(X, Y, Z) = aX^2 + bXY + cY^2 + dXZ + eYZ + fZ^2;$$

the correspondence/ˌkɒr.ɪˈspɒn.dəns/ 对应关系 is easy to understand as a recipe, or you can think of it as the bijection/baɪˈdʒek.ʃən/ 双射 $q \leftrightarrow Q$ given by

$$q(x, y) = Q(X/Z,\, Y/Z,\, 1) \quad \text{with } x = X/Z,\; y = Y/Z$$

and inversely,

$$Q = Z^2\, q(X/Z,\, Y/Z).$$
非齐次二次多项式 $q(x,y) = ax^2 + bxy + cy^2 + dx + ey + f$ 对应于齐次二次式 $Q(X,Y,Z) = aX^2 + bXY + cY^2 + dXZ + eYZ + fZ^2$。这个对应关系可以直接当作公式来理解，也可以把它看成双射 $q \leftrightarrow Q$：令 $x = X/Z$, $y = Y/Z$ 则 $q(x,y) = Q(X/Z, Y/Z, 1)$，反过来 $Q = Z^2\, q(X/Z, Y/Z)$。

A conic/ˈkɒn.ɪk/ 二次曲线 $C \subset \mathbb{P}^2$ is the curve given by $C : (Q(X, Y, Z) = 0)$, where $Q$ is a homogeneous quadratic expression; note that the condition $Q(X, Y, Z) = 0$ is well defined on the equivalence/ɪˈkwɪv.əl.əns/ 等价 class, since $Q(\lambda \mathbf{X}) = \lambda^2 Q(\mathbf{X})$ for any $\lambda \in \mathbb{R}$. As an exercise, check that the projective curve $C$ meets the affine/əˈfaɪn/ 仿射的 piece $\mathbb{R}^2$ in the affine conic given by $(q = 0)$.

二次曲线 $C \subset \mathbb{P}^2$ 是由 $C : (Q(X,Y,Z) = 0)$ 给出的曲线，其中 $Q$ 是齐次二次表达式。注意条件 $Q(X,Y,Z)=0$ 在等价类上是良定义的，因为对任何 $\lambda \in \mathbb{R}$ 都有 $Q(\lambda \mathbf{X}) = \lambda^2 Q(\mathbf{X})$。作为练习，请验证射影曲线 $C$ 与仿射片 $\mathbb{R}^2$ 的交就是由 $(q = 0)$ 给出的仿射二次曲线。
