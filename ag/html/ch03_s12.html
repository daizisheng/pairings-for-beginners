3.12 Finite algebras

3.12 有限代数

I now start on the proof of (3.8). Let $A \subset B$ be rings. As usual, $B$ is said to be finitely generated over $A$ (or f.g. as $A$-algebra) if there exist finitely many elements $b_1, \dots, b_n$ such that $B = A[b_1, \dots, b_n]$, so that $B$ is generated as a ring by $A$ and $b_1, \dots, b_n$.

现在我开始证明 (3.8)。设 $A \subset B$ 是环。照例，$B$ 称为在 $A$ 上有限生成的（或作为 $A$-代数是有限生成的），是指存在有限多个元素 $b_1, \dots, b_n$ 使得 $B = A[b_1, \dots, b_n]$，即 $B$ 作为环由 $A$ 和 $b_1, \dots, b_n$ 生成。

Contrast with the following definition: $B$ is a finite $A$-algebra if there exist finitely many elements $b_1, \dots, b_n$ such that $B = Ab_1 + \cdots + Ab_n$, that is, $B$ is finitely generated as $A$-module. The crucial/ˈkruː.ʃəl/ 关键的 distinction here is between generation as ring (when you're allowed any polynomial expressions in the $b_i$), and as module (the $b_i$ can only occur linearly/ˈlɪn.i.ə.li/ 线性地). For example, $k[X]$ is a finitely generated $k$-algebra (it's generated by one element $X$), but is not a finite $k$-algebra (since it has infinite dimension as $k$-vector space).

对比以下定义：$B$ 是一个有限 $A$-代数，是指存在有限多个元素 $b_1, \dots, b_n$ 使得 $B = Ab_1 + \cdots + Ab_n$，即 $B$ 作为 $A$-模是有限生成的。这里的关键区别在于：作为环生成（允许 $b_i$ 的任意多项式表达式），还是作为模生成（$b_i$ 只能线性出现）。例如，$k[X]$ 是有限生成的 $k$-代数（由一个元素 $X$ 生成），但不是有限 $k$-代数（因为它作为 $k$-向量空间是无限维的）。

Proposition

(i) Let $A \subset B \subset C$ be rings; then

$B$ a finite $A$-algebra and $C$ a finite $B$-algebra $\implies$ $C$ a finite $A$-algebra.

(ii) If $A \subset B$ is a finite $A$-algebra and $x \in B$ then $x$ satisfies a monic/ˈmɒn.ɪk/ 首一的 equation over $A$, that is, there exists a relation

$$x^n + a_{n-1}x^{n-1} + \cdots + a_0 = 0 \quad \text{with } a_i \in A$$

(note that the leading coefficient/ˌkəʊ.ɪˈfɪʃ.ənt/ 系数 is 1).

(iii) Conversely, if $x$ satisfies a monic equation over $A$, then $B = A[x]$ is a finite $A$-algebra.

命题
(i) 设 $A \subset B \subset C$ 是环；则 $B$ 是有限 $A$-代数且 $C$ 是有限 $B$-代数 $\implies$ $C$ 是有限 $A$-代数。
(ii) 如果 $A \subset B$ 是有限 $A$-代数，$x \in B$，则 $x$ 满足一个 $A$ 上的首一方程，即存在关系式 $x^n + a_{n-1}x^{n-1} + \cdots + a_0 = 0$，其中 $a_i \in A$（注意首项系数为1）。
(iii) 反过来，如果 $x$ 满足 $A$ 上的一个首一方程，则 $B = A[x]$ 是有限 $A$-代数。

Proof   (i) and (iii) are easy exercises (compare similar results for field extensions). For (ii), I use a rather nonobvious/nɒnˈɒb.vi.əs/ 不显然的 'determinant trick' (I didn't think of it for myself): suppose $B = \sum Ab_i$; for each $i$, $xb_i \in B$, so there exist constants $a_{ij} \in A$ such that

$$xb_i = \sum_j a_{ij} b_j.$$

This can be written

$$\sum_j (x\delta_{ij} - a_{ij})b_j = 0,$$

where $\delta_{ij}$ is the identity matrix. Now let $M$ be the matrix with

$$M_{ij} = x\delta_{ij} - a_{ij},$$

and set $\Delta = \det M$. Then by standard linear algebra, (writing $\mathbf{b}$ for the column vector with entries $(b_1, \dots, b_n)$ and $M^{\mathrm{adj}}$ for the adjoint/ˈæd.ʒɔɪnt/ 伴随的 matrix of $M$),

$$M\mathbf{b} = 0, \quad \text{hence} \quad 0 = (M^{\mathrm{adj}})M\mathbf{b} = \Delta\mathbf{b},$$

and therefore $\Delta b_i = 0$ for all $i$. However, $1_B \in B$ is a linear combination of the $b_i$, so that $\Delta = \Delta \cdot 1_B = 0$, and I've won my relation:

$$\det(x\delta_{ij} - a_{ij}) = 0.$$

This is obviously a monic relation for $x$ with coefficients in $A$.   Q.E.D.

证明   (i) 和 (iii) 是简单的练习（类比域扩张的类似结论）。对于 (ii)，我使用一个相当不显然的"行列式技巧"（不是我自己想出来的）：假设 $B = \sum Ab_i$；对每个 $i$，$xb_i \in B$，所以存在常数 $a_{ij} \in A$ 使得 $xb_i = \sum_j a_{ij} b_j$。这可以写成 $$\sum_j (x\delta_{ij} - a_{ij}) b_j = 0,$$ 其中 $\delta_{ij}$ 是单位矩阵。令 $M$ 为矩阵 $M_{ij} = x\delta_{ij} - a_{ij}$，令 $\Delta = \det M$。由标准线性代数（记 $\mathbf{b}$ 为列向量 $(b_1, \dots, b_n)$，$M^{\mathrm{adj}}$ 为 $M$ 的伴随矩阵）， $$M\mathbf{b} = 0, \quad \text{从而} \quad 0 = (M^{\mathrm{adj}})M\mathbf{b} = \Delta\mathbf{b},$$ 因此 $\Delta b_i = 0$ 对所有 $i$ 成立。但 $1_B \in B$ 是 $b_i$ 的线性组合，所以 $\Delta = \Delta \cdot 1_B = 0$，于是我就得到了关系式： $$\det(x\delta_{ij} - a_{ij}) = 0.$$ 这显然是 $x$ 在 $A$ 上的首一关系式。证毕。
