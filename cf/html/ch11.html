11.1 Harmonic Functions

11.1 调和函数

A function \( u(x,y) \) is harmonic if \( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0 \). The real and imaginary parts of analytic functions are harmonic.

如果 \( \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0 \),则函数 \( u(x,y) \) 是调和的。解析函数的实部和虚部都是调和的。

11.2 The Dirichlet Problem

11.2 狄利克雷问题

The Dirichlet problem is to find a harmonic function in a region with prescribed boundary values. This has important applications in physics and engineering.

狄利克雷问题是在具有规定边界值的区域中找到调和函数。这在物理和工程中有重要应用。
