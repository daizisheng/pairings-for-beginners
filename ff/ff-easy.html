1. 从整数到复数：数学家为什么要扩张数系？

核心问题
我们从小就认识整数、分数、小数……但数学家为什么要发明这么多不同的「数」？答案很简单：为了解方程。每一次数系的扩张，都是因为在原来的数系里，某些「合理的」方程无解。

1.1 整数：加法的天堂

让我们从最熟悉的整数 $\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$ 开始。

整数有一个美妙的性质：对于加法，整数是完美的。具体来说：

  • 封闭性：任意两个整数相加，结果还是整数
  • 结合律：$(a + b) + c = a + (b + c)$
  • 单位元：存在 $0$，使得 $a + 0 = a$
  • 逆元：每个整数 $a$ 都有「相反数」$-a$，使得 $a + (-a) = 0$
  • 交换律：$a + b = b + a$

在代数学中，满足前四条性质的结构叫做群（group），如果还满足交换律，就叫做阿贝尔群（Abelian group）或交换群。所以我们说：

关键观察 1
整数 $\mathbb{Z}$ 关于加法构成一个阿贝尔群。

但整数不仅有加法，还有乘法！乘法也满足封闭性、结合律、有单位元（$1$）、以及对加法的分配律。同时具备加法和乘法两种运算、且满足这些性质的结构，叫做环（ring）。

关键观察 2
整数 $\mathbb{Z}$ 是一个环 —— 这比单纯说它是一个群更加精确地描述了整数的代数结构。

1.2 从整数到有理数：除法的需求

整数环有一个「缺陷」：除法做不了。

比如，方程 $2x = 1$ 在整数中无解 —— 没有哪个整数乘以 $2$ 等于 $1$。换句话说，$2$ 在整数中没有乘法逆元。

需求：乘法逆元
我们希望每个非零的数 $a$ 都有一个「倒数」$a^{-1}$，使得 $a \cdot a^{-1} = 1$。这样，方程 $ax = b$ 就总有解 $x = a^{-1}b$。

为了满足这个需求，我们必须扩张整数。最小的扩张是什么？就是有理数：

$$\mathbb{Q} = \left\{ \frac{a}{b} : a, b \in \mathbb{Z}, b \neq 0 \right\}$$

在有理数中，每个非零元素 $\frac{a}{b}$ 都有乘法逆元 $\frac{b}{a}$。一个环，如果它的每个非零元素都有乘法逆元，就叫做域（field）。

关键观察 3
有理数 $\mathbb{Q}$ 是包含整数 $\mathbb{Z}$ 的最小的域。数学家称 $\mathbb{Q}$ 为 $\mathbb{Z}$ 的分式域（field of fractions）。

现在，所有形如 $ax = b$（$a \neq 0$）的一次方程都有解了。但数学家的胃口不止于此 —— 他们想解的是所有多项式方程。

1.3 从有理数到实数：无理数的出现

考虑这个简单的方程：

$$x^2 = 2$$

这个方程在有理数中有解吗？

经典证明：$\sqrt{2}$ 是无理数

假设 $\sqrt{2} = \frac{p}{q}$，其中 $p, q$ 是互素的整数。

则 $2 = \frac{p^2}{q^2}$，即 $p^2 = 2q^2$。

所以 $p^2$ 是偶数，因此 $p$ 是偶数，设 $p = 2k$。

代入得 $4k^2 = 2q^2$，即 $q^2 = 2k^2$，所以 $q$ 也是偶数。

但这与 $p, q$ 互素矛盾！因此 $\sqrt{2}$ 不是有理数。

这就尴尬了：一个系数全是有理数（甚至是整数）的方程，在有理数中居然没有解！

为了解决这个问题，我们需要引入无理数（irrational numbers），比如 $\sqrt{2}$、$\sqrt{3}$、$\pi$、$e$ 等。有理数和无理数一起，构成了实数 $\mathbb{R}$。

几何直观
实数可以和数轴上的点一一对应。有理数只占据了数轴上「稀疏」的一些点（虽然它们稠密地分布），而无理数填满了所有的「空隙」。数学上说，$\mathbb{R}$ 是 $\mathbb{Q}$ 的完备化（completion）。

现在 $x^2 = 2$ 有解了（$x = \pm\sqrt{2}$）。实数是不是就够用了呢？

1.4 从实数到复数：负数开方的需求

考虑这个方程：

$$x^2 + 1 = 0 \quad \text{即} \quad x^2 = -1$$

这个方程在实数中有解吗？

没有。因为任何实数的平方都是非负的： $$x^2 \geq 0 \quad \text{对所有 } x \in \mathbb{R}$$ 所以 $x^2 = -1$ 在实数中无解。

怎么办？数学家的解决方案简单粗暴：假装它有解。

我们「发明」一个新的数 $i$，定义它满足 $i^2 = -1$，然后把它和实数混在一起，得到复数：

$$\mathbb{C} = \{ a + bi : a, b \in \mathbb{R} \}$$
复数的运算

加法：$(a + bi) + (c + di) = (a+c) + (b+d)i$

乘法：$(a + bi)(c + di) = ac + adi + bci + bdi^2 = (ac - bd) + (ad + bc)i$

（利用 $i^2 = -1$）

可以验证，复数 $\mathbb{C}$ 也是一个域。现在 $x^2 + 1 = 0$ 有解了：$x = \pm i$。

但这里出现了一个深刻的问题：我们会不会需要一直这样扩张下去？会不会有某个方程在 $\mathbb{C}$ 中也没有解，迫使我们再发明新的数？

1.5 代数基本定理：扩张的终点

答案是：不需要了。复数是这场扩张游戏的终点。

代数基本定理（Fundamental Theorem of Algebra）

每一个次数 $\geq 1$ 的复系数多项式，在 $\mathbb{C}$ 中至少有一个根。

等价地说：每一个 $n$ 次复系数多项式，在 $\mathbb{C}$ 中恰好有 $n$ 个根（计重数）。

这个定理告诉我们一个惊人的事实：虽然复数是为了解 $x^2 + 1 = 0$ 而发明的，但它「免费」地解决了所有多项式方程！

例子

$x^2 + 1 = 0$ 的根：$x = \pm i$

$x^3 - 1 = 0$ 的根：$x = 1, \frac{-1+\sqrt{3}i}{2}, \frac{-1-\sqrt{3}i}{2}$（三个三次单位根）

$x^4 + 1 = 0$ 的根：$x = \frac{\pm 1 \pm i}{\sqrt{2}}$（四个八次单位根中的一半）

一个域，如果它的每个非常数多项式都在这个域中有根，就称为代数闭域（algebraically closed field）。

本节总结：数系扩张的历程

关键结论：复数 $\mathbb{C}$ 是有理数 $\mathbb{Q}$（乃至实数 $\mathbb{R}$）的代数闭包（algebraic closure）。在 $\mathbb{C}$ 中，所有以 $\mathbb{Q}$（或 $\mathbb{R}$）为系数的多项式方程都有解，而且所有的解也都在 $\mathbb{C}$ 中。

下一个问题

复数很美好，但它是无限的。在计算机科学和密码学中，我们常常需要在有限的世界里工作。

那么问题来了：有没有「有限的域」？它们长什么样？它们也有「代数闭包」吗？

2. 最简单的有限域 $\mathbb{F}_2$

2.1 $\mathbb{F}_2$ 是一个域

最小的素数是 $2$，所以我们从 $\mathbb{F}_2$ 开始。它只有两个元素：

$$\mathbb{F}_2 = \{0, 1\}$$

运算规则就是普通的加法和乘法，但结果要对 $2$ 取模：

$+$$0$$1$
$0$$0$$1$
$1$$1$$0$
$\times$$0$$1$
$0$$0$$0$
$1$$0$$1$

验证域的条件：

  • 加法单位元：$0$
  • 乘法单位元：$1$
  • 加法逆元：$0$ 的逆元是 $0$，$1$ 的逆元是 $1$（因为 $1 + 1 = 0$）
  • 乘法逆元：$1$ 的逆元是 $1$（只需要非零元素有逆元）

穷举完毕，$\mathbb{F}_2$ 确实是一个域。

2.2 $\mathbb{F}_2$ 是代数闭的吗？

回忆第一节的问题：复数 $\mathbb{C}$ 之所以是「终点」，是因为所有多项式在 $\mathbb{C}$ 中都有根。$\mathbb{F}_2$ 有这个性质吗？

考虑 $\mathbb{F}_2$ 上的多项式 $f(x) = x^2 + x + 1$。检验它在 $\mathbb{F}_2$ 中的值：

$$f(0) = 0 + 0 + 1 = 1 \neq 0$$ $$f(1) = 1 + 1 + 1 = 1 \neq 0$$

所以 $x^2 + x + 1$ 在 $\mathbb{F}_2$ 中没有根。

关键发现
$\mathbb{F}_2$ 不是代数闭的。存在 $\mathbb{F}_2$ 系数的多项式，在 $\mathbb{F}_2$ 中无解。

这和 $x^2 = 2$ 在 $\mathbb{Q}$ 中无解、$x^2 = -1$ 在 $\mathbb{R}$ 中无解是一样的情况。让我们更系统地看看 $\mathbb{F}_2$ 上的多项式的根的情况。

2.4 不可约多项式：扩张的钥匙

从上面的穷举中，我们关注那些不可约的多项式——即不能再因式分解的。

为什么一次多项式不用考虑？因为一次多项式 $x - a$ 在任何域中都有根 $a$。这是显然的：把 $a$ 代入就得到 $a - a = 0$。

那高次不可约多项式呢？如果 $f(x)$ 不可约且 $\deg f \geq 2$，那么 $f$ 在 $\mathbb{F}_2$ 中一个根也没有。为什么？因为如果 $f(\beta) = 0$，那就意味着 $(x - \beta)$ 整除 $f(x)$，$f$ 就可以分解了——矛盾。

关键观察
不可约多项式 = 在当前域中完全无根的多项式（对于次数 $\geq 2$ 的情况）。它们就像 $x^2 - 2$ 对于 $\mathbb{Q}$、$x^2 + 1$ 对于 $\mathbb{R}$ 一样，是扩张域的「种子」。

把 $\mathbb{F}_2$ 上次数 $\leq 4$ 的不可约多项式汇总：

次数不可约多项式个数
$1$$x$，$x+1$$2$
$2$$x^2+x+1$$1$
$3$$x^3+x+1$，$x^3+x^2+1$$2$
$4$$x^4+x+1$，$x^4+x^3+1$，$x^4+x^3+x^2+x+1$$3$
Remark 1
$x^4+x^2+1$ 虽然在 $\mathbb{F}_2$ 中无根，但它可约：$x^4+x^2+1 = (x^2+x+1)^2$。这种「无根但可约」的现象在讨论分裂域（splitting field）和可分性（separability）时很重要，但我们暂且不管。
Remark 2
我们只穷举到了次数 $\leq 4$。次数 $5, 6, 7, \ldots$ 的不可约多项式还有很多很多。

既然存在 2 次不可约多项式 $x^2+x+1$，$\mathbb{F}_2$ 肯定不是代数闭的。这引出一个自然的疑问：

疑问
对于任意的 $n$，$\mathbb{F}_p$ 上都存在 $n$ 次不可约多项式吗？

答案：是的。

$\mathbb{F}_p$ 上 $n$ 次首一不可约多项式的个数记为 $I_p(n)$，有公式：

$$I_p(n) = \frac{1}{n} \sum_{d \mid n} \mu(n/d) \cdot p^d$$

其中 $\mu$ 是 Möbius 函数。主项是 $\frac{p^n}{n}$，其他项的贡献相对很小，所以 $I_p(n) > 0$ 对所有 $n \geq 1$ 成立。

直观理解：$\mathbb{F}_p$ 上 $n$ 次首一多项式共有 $p^n$ 个（$n$ 个系数各有 $p$ 种选择）。其中大约 $\frac{1}{n}$ 的比例是不可约的。

验证：$\mathbb{F}_2$ 上的情况

$I_2(1) = 2$（即 $x$ 和 $x+1$）

$I_2(2) = \frac{1}{2}(2^2 - 2^1) = 1$（即 $x^2+x+1$）

$I_2(3) = \frac{1}{3}(2^3 - 2^1) = 2$（即 $x^3+x+1$ 和 $x^3+x^2+1$）

$I_2(4) = \frac{1}{4}(2^4 - 2^2) = 3$（即 $x^4+x+1$，$x^4+x^3+1$，$x^4+x^3+x^2+x+1$）

与我们穷举的结果完全一致！

证明思路（选读）

考虑多项式 $x^{p^n} - x$。它恰好等于所有次数 $d \mid n$ 的不可约多项式的乘积。为什么？

  • 首先，$x^{p^n} - x$ 没有重根（求导得 $p^n x^{p^{n}-1} - 1 = -1$，与原式互素）
  • 其次，若 $f(x)$ 是 $d$ 次不可约多项式且 $d \mid n$，则 $f$ 的根 $\beta$ 满足 $\beta^{p^n} = \beta$（因为 $\beta^{p^d} = \beta$ 且 $d \mid n$），所以 $f$ 的所有根都是 $x^{p^n} - x$ 的根
  • 因为没有重根，所以 $f \mid (x^{p^n} - x)$

比较次数得：$p^n = \sum_{d \mid n} d \cdot I_p(d)$。用 Möbius 反演公式「解」这个递推，就得到 $I_p(n)$ 的显式公式。

结论
$\mathbb{F}_p$ 不是代数闭的，而且「差得很远」——对每个 $n$ 都有大量的 $n$ 次不可约多项式等着我们去处理。

2.5 第一次扩张：从 $\mathbb{F}_2$ 到 $\mathbb{F}_4$

类比从 $\mathbb{R}$ 到 $\mathbb{C}$ 的扩张：我们假装 $x^2 + 1 = 0$ 有一个根叫 $i$，然后把 $i$ 和实数混在一起。

现在对 $\mathbb{F}_2$ 做同样的事：假设 $x^2 + x + 1 = 0$ 有一个根叫 $\beta$。把 $\beta$ 和 $\mathbb{F}_2$ 的元素混在一起。

为什么只需要考虑 $\{a + b\beta\}$ 的形式？因为 $\beta$ 满足 $\beta^2 + \beta + 1 = 0$，即 $\beta^2 = \beta + 1$。所以 $\beta$ 的高次幂都可以化简：

  • $\beta^2 = \beta + 1$
  • $\beta^3 = \beta \cdot \beta^2 = \beta(\beta + 1) = \beta^2 + \beta = (\beta + 1) + \beta = 1$
  • $\beta^4 = \beta \cdot \beta^3 = \beta$
  • ……

所有元素都可以写成 $a + b\beta$（$a, b \in \mathbb{F}_2$）的形式，因此：

$$\mathbb{F}_2(\beta) = \{a + b\beta : a, b \in \mathbb{F}_2\} = \{0, 1, \beta, 1+\beta\}$$

这个新的集合有 $4$ 个元素。运算规则：

  • 加法：和 $\mathbb{F}_2$ 一样，系数对 $2$ 取模
  • 乘法：正常展开，然后用 $\beta^2 = \beta + 1$（从 $\beta^2 + \beta + 1 = 0$ 得到）化简

$0$$0$$1$$\beta$$1+\beta$

$\beta$$\beta$$1+\beta$$0$$1$

验证 $\beta \cdot (1+\beta) = \beta + \beta^2 = \beta + (\beta + 1) = 1$。所以 $\beta$ 的乘法逆元是 $1+\beta$。

结果
$\mathbb{F}_2(\beta)$ 是一个有 $4$ 个元素的域，记作 $\mathbb{F}_4$。

$x^2 + x + 1$ 在 $\mathbb{F}_4$ 中的命运：现在它有根了：

  • $\beta^2 + \beta + 1 = 0$ ✓（这是我们构造 $\beta$ 的方式）
  • $(1+\beta)^2 + (1+\beta) + 1 = (1 + \beta^2) + 1 + \beta + 1 = 1 + (\beta+1) + 1 + \beta + 1 = 0$ ✓

所以 $x^2 + x + 1$ 在 $\mathbb{F}_4$ 上可以分解：

$$x^2 + x + 1 = (x + \beta)(x + (1+\beta)) = (x - \beta)(x - (1+\beta))$$

（注意：在 $\mathbb{F}_2$ 及其扩张中，$-1 = 1$，所以 $x + \beta = x - \beta$）

$\mathbb{F}_4$ 是代数闭的吗？我们解决了 $x^2 + x + 1$，但 $\mathbb{F}_4$ 上的多项式比 $\mathbb{F}_2$ 上多得多——系数可以取 $4$ 个值了！

先看看原来 $\mathbb{F}_2$ 上的那些不可约多项式，在 $\mathbb{F}_4$ 上怎么样了：

$x^2+x+1$$\beta$, $1+\beta$$(x+\beta)(x+1+\beta)$

$x^3+x^2+1$无不可约
$x^4+x+1$无$(x^2+x+\beta)(x^2+x+1+\beta)$
$x^4+x^3+1$无不可约
$x^4+x^3+x^2+x+1$无？（待验证）

验证 $x^3 + x + 1$ 在 $\mathbb{F}_4$ 中无根（先算 $\beta^3 = \beta \cdot \beta^2 = \beta(\beta+1) = \beta^2 + \beta = 1$）：

  • $f(\beta) = \beta^3 + \beta + 1 = 1 + \beta + 1 = \beta \neq 0$
  • $f(1+\beta) = (1+\beta)^3 + (1+\beta) + 1 = 1 + 1 + \beta + 1 = 1 + \beta \neq 0$

$x^3 + x + 1$ 是 $3$ 次多项式，在 $\mathbb{F}_4$ 中无根意味着它在 $\mathbb{F}_4$ 上仍然不可约（$3$ 次多项式若可约必有一次因子，即有根）。

结论

$\mathbb{F}_4$ 也不是代数闭的。$x^3 + x + 1$ 等多项式在 $\mathbb{F}_4$ 中仍然无根。

我们需要继续扩张……

有趣的现象
$x^4 + x + 1$ 在 $\mathbb{F}_2$ 上不可约，但在 $\mathbb{F}_4$ 上变得可约了（分解成两个 $2$ 次因子），尽管它在 $\mathbb{F}_4$ 中没有根！这再次说明：无根 $\neq$ 不可约。

2.6 $\mathbb{F}_4$ 上的多项式：数量爆炸

$\mathbb{F}_4$ 有 $4$ 个元素，所以 $n$ 次首一多项式有 $4^n$ 个（$n$ 个系数各有 $4$ 种选择）：

  • $\mathbb{F}_4$ 仍然不是代数闭的

2.7 共轭与同构：选择不同的根会怎样？

扩张过程似乎无穷无尽，我们先暂停，回头审视刚才的扩张。

$x^2 + x + 1 = 0$ 有两个根：$\beta$ 和 $1+\beta$。我们选择了 $\beta$ 来扩张。但如果选择 $1+\beta$ 呢？

设 $\gamma = 1 + \beta$。$\gamma$ 也满足 $\gamma^2 + \gamma + 1 = 0$（可以验证），所以 $\gamma^2 = \gamma + 1$。

Frobenius 自同构

映射 $\varphi: x \mapsto x^2$ 把 $\beta$ 映到 $\beta^2 = 1+\beta = \gamma$。这个映射叫做 Frobenius 自同构，它把每个元素映到它的「共轭」。在有限域理论中，Frobenius 映射扮演着核心角色。

2.8 Galois 群：对称性的代数描述

上一节我们发现，$\beta$ 和 $\gamma = 1+\beta$ 可以「互换」而不破坏域的结构。这种「可互换性」正是 Galois 群所描述的对象。

定义：域扩张 $\mathbb{F}_4 / \mathbb{F}_2$ 的 Galois 群，记作 $\mathrm{Gal}(\mathbb{F}_4 / \mathbb{F}_2)$，是所有满足以下条件的映射 $\sigma: \mathbb{F}_4 \to \mathbb{F}_4$ 构成的集合：

  1. $\sigma$ 是自同构：保持加法和乘法，即 $\sigma(x+y) = \sigma(x)+\sigma(y)$，$\sigma(xy) = \sigma(x)\sigma(y)$
  2. $\sigma$ 固定基域：对所有 $a \in \mathbb{F}_2$，有 $\sigma(a) = a$

找出所有这样的映射：

设 $\sigma$ 是 $\mathbb{F}_4$ 的一个自同构且固定 $\mathbb{F}_2$。$\sigma$ 由 $\sigma(\beta)$ 完全决定（因为 $\sigma(a + b\beta) = a + b\sigma(\beta)$）。

$\sigma(\beta)$ 必须满足什么条件？因为 $\beta$ 是 $x^2 + x + 1 = 0$ 的根，而 $\sigma$ 保持运算：

$$\sigma(\beta)^2 + \sigma(\beta) + 1 = \sigma(\beta^2 + \beta + 1) = \sigma(0) = 0$$

所以 $\sigma(\beta)$ 也必须是 $x^2 + x + 1 = 0$ 的根。但这个方程只有两个根：$\beta$ 和 $\gamma = 1+\beta$。

因此，恰好有两个自同构：

自同构$\sigma(0)$$\sigma(1)$$\sigma(\beta)$$\sigma(1+\beta)$描述
$\mathrm{id}$$0$$1$$\beta$$1+\beta$恒等映射
$\sigma$$0$$1$$1+\beta$$\beta$Frobenius: $x \mapsto x^2$

验证 $\sigma(x) = x^2$ 确实把 $\beta$ 映到 $1+\beta$：$\sigma(\beta) = \beta^2 = \beta + 1$ ✓

群结构：

这两个映射在复合运算下构成一个群：

$\circ$$\mathrm{id}$$\sigma$
$\mathrm{id}$$\mathrm{id}$$\sigma$
$\sigma$$\sigma$$\mathrm{id}$

验证 $\sigma \circ \sigma = \mathrm{id}$：$\sigma(\sigma(\beta)) = \sigma(1+\beta) = (1+\beta)^2 = 1 + \beta^2 = 1 + \beta + 1 = \beta$ ✓

结论
$$\mathrm{Gal}(\mathbb{F}_4 / \mathbb{F}_2) = \{\mathrm{id}, \sigma\} \cong \mathbb{Z}/2\mathbb{Z}$$

Galois 群的阶 $= 2 = [\mathbb{F}_4 : \mathbb{F}_2]$（扩张次数）。这不是巧合！

固定点：

哪些元素被 $\sigma$ 固定（即 $\sigma(x) = x$）？

  • $\sigma(0) = 0$ ✓
  • $\sigma(1) = 1$ ✓
  • $\sigma(\beta) = 1+\beta \neq \beta$
  • $\sigma(1+\beta) = \beta \neq 1+\beta$

被 $\sigma$ 固定的元素恰好是 $\{0, 1\} = \mathbb{F}_2$——基域！

Galois 对应

Galois 群的「固定域」恰好是基域 $\mathbb{F}_2$。这是 Galois 理论的核心：

子群 $\longleftrightarrow$ 中间域

在这个简单例子中：$\{\mathrm{id}\}$ 对应 $\mathbb{F}_4$，$\{\mathrm{id}, \sigma\}$ 对应 $\mathbb{F}_2$。

为什么叫「Galois」群？

Évariste Galois（伽罗瓦，1811–1832）发现了多项式根的「对称性」与方程可解性之间的深刻联系。他在决斗中去世，年仅 20 岁。Galois 群正是以他命名，用来描述「哪些根的置换保持代数关系不变」。

2.9 用 $x^3 + x + 1$ 扩张：构造 $\mathbb{F}_8$

$\xi^4 = 1+\xi+\xi^2$$\mapsto$$\omega^{12} = \omega^5 = 1+\omega+\omega^2$
$\xi^5 = 1+\xi$$\mapsto$$\omega^{15} = \omega^1 = \omega$
$\xi^6 = \xi+\xi^2$$\mapsto$$\omega^{18} = \omega^4 = \omega+\omega^2$

验证 $\varphi$ 保持乘法：$\varphi(\xi^3) = \varphi(1+\xi^2)$？

  • 左边：$\varphi(\xi^3) = \omega^9 = \omega^2$
  • 右边：$\varphi(1) + \varphi(\xi^2) = 1 + \omega^6 = 1 + (1+\omega^2) = \omega^2$ ✓
核心结论

$\mathbb{F}_2(\omega) \cong \mathbb{F}_2(\xi)$——用不同的不可约多项式扩张，得到的域是同构的！

$\mathbb{F}_8$ 在同构意义下是唯一的：不管用哪个 3 次不可约多项式来构造，最终得到的都是「同一个」域。

更一般的结论

对于任意素数 $p$ 和正整数 $n$，$p^n$ 元有限域 $\mathbb{F}_{p^n}$ 在同构意义下是唯一的。不同的 $n$ 次不可约多项式给出同构的域。

证明思路：设 $K$ 是任意 $p^n$ 元域。

  1. $K^*$ 是 $p^n - 1$ 阶循环群，所以 $K$ 中每个元素都满足 $x^{p^n} = x$
  2. 即 $K$ 的元素恰好是 $x^{p^n} - x$ 在 $\overline{\mathbb{F}_p}$ 中的全部根
  3. $x^{p^n} - x$ 的分裂域（包含所有根的最小域）是唯一的（在同构意义下）
  4. 而这个分裂域恰好就是 $K$

所以所有 $p^n$ 元域都同构于 $x^{p^n} - x$ 的分裂域——它们是「同一个」域。

这就是为什么我们可以直接写「$\mathbb{F}_8$」而不用指明是用哪个多项式构造的——因为结果都一样！

重要推论

$\mathbb{F}_{p^d}$ 包含所有 $\mathbb{F}_p$ 上 $d$ 次不可约多项式的根！

为什么？

  1. 设 $f(x)$ 是任意 $d$ 次不可约多项式，$\alpha$ 是它的一个根
  2. $\alpha$ 生成的域 $\mathbb{F}_p(\alpha)$ 有 $p^d$ 个元素
  3. 由唯一性，$\mathbb{F}_p(\alpha) \cong \mathbb{F}_{p^d}$
  4. 所以 $\alpha \in \mathbb{F}_{p^d}$

换句话说：用 $x^3+x+1$ 构造的 $\mathbb{F}_8$ 自动包含 $x^3+x^2+1$ 的根——不用再「添加」，它们已经在里面了！

验证：$x^3+x^2+1$ 的根在 $\mathbb{F}_2(\omega)$ 中

我们已经算过：$x^3+x^2+1$ 在 $\mathbb{F}_2(\omega)$ 中的三个根是 $\omega^3$、$\omega^5$、$\omega^6$。

它们确实都在 $\mathbb{F}_8 = \{0, 1, \omega, \omega^2, \omega^3, \omega^4, \omega^5, \omega^6\}$ 中！

更深的理解

$x^{p^d} - x$ 恰好等于 $\mathbb{F}_p$ 上所有次数整除 $d$ 的不可约多项式的乘积。所以 $\mathbb{F}_{p^d}$（作为 $x^{p^d}-x$ 的分裂域）自然包含了这些多项式的所有根。

同构映射的选择

$\xi$ 可以映到 $x^3+x^2+1=0$ 在 $\mathbb{F}_2(\omega)$ 中的任意一个根。这三个根是 $\omega^3$、$\omega^5$、$\omega^6$（互为共轭），对应三个不同的同构映射。这正是 $|\mathrm{Gal}(\mathbb{F}_8/\mathbb{F}_2)| = 3$。
